# Password Generator 🔐

This is a simple React-based password generator.  
It allows you to generate strong passwords with:

- Customizable length
- Numbers
- Special characters

## Features

✅ Random password generation  
✅ Copy to clipboard  
✅ Responsive UI with Tailwind CSS  

## Tech Stack

- React
- Tailwind CSS
- JavaScript
